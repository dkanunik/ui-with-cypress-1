# Cypress sandbox project

##### ENV variables
```
FINANCE_BASE_HREF=https://finance.yahoo.com
```

##### Run tests on CI
```
npm run cy:ui
```

##### Run tests on localhost 
```
set CYPRESS_FINANCE_BASE_HREF=https://finance.yahoo.com
npm run cy:open
```

##### Open Allure report
```
npm run allure-report-open
```
